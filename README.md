# bubble-game

A simple browser-based bubble game where players must quickly hit the bubble matching the target number shown in the "Hit" section before time runs out.

---


The Bubble Hit Game is a fun and interactive JavaScript game built using HTML, CSS, and vanilla JavaScript. Randomly generated bubbles appear on the screen, each containing a number. The player must click the correct bubble based on the target number displayed.

---

- Random number bubbles generated dynamically
- Target "Hit" number updates after each correct hit
- Score tracking system
- Countdown timer
- Game over screen when time runs out
